# Support Ticket Classification & Prioritization

## Objective
To build a Machine Learning model that automatically classifies customer support tickets into categories and helps support teams prioritize customer issues.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Dataset
Customer Support Tickets Dataset

## Project Workflow
1. Data Loading
2. Data Exploration
3. Missing Value Analysis
4. Text Feature Extraction using TF-IDF
5. Train-Test Split
6. Ticket Classification using Multinomial Naive Bayes
7. Model Evaluation
8. Ticket Category and Priority Analysis

## Model Performance
- Accuracy: 19.13%

## Key Features
- Ticket Category Classification
- Priority Analysis
- TF-IDF Vectorization
- NLP-based Processing
- Performance Evaluation

## Business Value
This project helps organizations:
- Automate ticket categorization
- Improve support team efficiency
- Prioritize critical customer issues
- Reduce manual ticket handling effort
- Improve response times and customer satisfaction
